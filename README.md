# WatchBase Web

Landing page and privacy policy for the WatchBase iOS app.

## Repository

This repository contains the GitHub Pages site for WatchBase, including:
- Landing page with app information and App Store link
- Privacy Policy page required for App Store Connect

## GitHub Pages

This site is automatically deployed to GitHub Pages. The site is available at:
https://joshgallantt.github.io/WatchBaseWeb/

## Local Development

To view the site locally, simply open `index.html` in a web browser or use a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.

## Structure

- `index.html` - Main landing page
- `privacy-policy.html` - Privacy policy page
- `README.md` - This file
